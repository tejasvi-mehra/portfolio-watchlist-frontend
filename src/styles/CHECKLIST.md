# Styles Checklist (`src/styles`)

## Mission

Define design tokens and animation rules that support realtime readability and performance.

## Must implement

- [ ] Color semantics for up/down flashes and neutral values.
- [ ] Consistent spacing/type scale across watchlist, portfolio, and asset detail.
- [ ] Lightweight transition presets for high-frequency numeric changes.
- [ ] Accessible contrast for dark/light themes.

## Strict directives

- Be factual, rigorous, strict.
- Push back on heavy animation techniques that risk frame drops.
- Prefer CSS transforms/opacities over expensive layout thrash patterns.
